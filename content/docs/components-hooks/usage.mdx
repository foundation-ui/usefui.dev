<DocsSection>
### Setup

<DocsTextBlock className="m-b-medium-60">
  `@usefui/components` are built using the [React Context
  API](https://react.dev/reference/react/useContext). When using Component
  Hooks, you need to wrap you component in a `ContextProvider` in order to
  access the states and methods exposed by the Component API.
</DocsTextBlock>

<DocsCodePreview
  language="tsx"
  code={`import { Dialog, useDialog } from "@usefui/components";

    function Provider() {
        return (
          <Dialog.Root>
            <Component />
          </Dialog.Root>
        );

    }

    function Component() {
        const dialog = useDialog();
        const { states, methods } = dialog;

        return ( {/** Your code here **/} );

    }

`} />
