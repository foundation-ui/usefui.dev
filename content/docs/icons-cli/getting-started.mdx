<DocsSection>
### Quick Start

<DocsTextBlock className="m-b-medium-60">
  To get started with `@usefui/svgjsx`, run the following command:
</DocsTextBlock>

#### Npm

<DocsCodePreview language="shell" code={`npx @usefui/svgjsx generate`} />

#### Pnpm

<DocsCodePreview language="shell" code={`pnpx @usefui/svgjsx generate`} />

#### Yarn

<DocsCodePreview language="shell" code={`yarn dlx @usefui/svgjsx generate`} />

#### Bun

<DocsCodePreview language="shell" code={`bunx @usefui/svgjsx generate`} />

</DocsSection>

<DocsBanner
  title="CLI default options"
  description="If you don't use the CLI options, the default input/output directories will be set to svg and icons at the root of your project, make sure you have the svg folder with your svg icon files in it before you run the generate command."
  mode="warning"
/>
