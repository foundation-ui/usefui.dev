### Install

<DocsTextBlock className="m-b-medium-60">
  To get the most out of `@usefui`, it is strongly recommended to use at least
  `@usefui/components` and `@usefui/tokens`.
</DocsTextBlock>

<DocsTextBlock className="m-b-medium-60">
  Run the following commands to install `@usefui` packages in project:
</DocsTextBlock>

<DocsCodePreview
  language="shell"
  code={`pnpm add @usefui/components @usefui/tokens @usefui/core @usefui/styles @usefui/analytics @usefui/hooks`}
/>
</DocsSection>

<DocsSection>
  ### Setup
  <DocsTextBlock>
    Foundation Design System is styled using CSS in JS with
    [styled-components](https://styled-components.com/). This technique, coupled
    with `@usefui/core`, requires a
    [GlobalStyle](https://styled-components.com/docs/faqs#note-regarding-css-import-and-createglobalstyle)
    definition to convert your Design Tokens Library as CSS variables.
     **_The styling can be done using styled-components or any other techniques/frameworks._**
  </DocsTextBlock>

  <br />
  <DocsTextBlock>
    Add the `ColorModeProvider` component to your setup to support the design tokens
    provided by `@usefui/tokens` or generated using `@usefui/core`.
    It enables you to switch between Themes based on user preferences.
  </DocsTextBlock>
  <br />
  <DocsTextBlock className="m-b-medium-60">
    `@usefui/tokens` comes with a set of pre-built GlobalStyles
    definitions to ensure a consistent visual language across all your apps,
    regardless of platform or device. You can build your own using `@usefui/core` and replace the default integration using your custom values.
  </DocsTextBlock>
  <DocsCodePreview
    code={`
      "use client"; // For nextjs apps

      import React from "react";
      import { ColorModeProvider } from "@usefui/tokens";

      import {
        ResetStyles,
        TypographyRoot,
        CSSRoot
      } from "@usefui/styles";

      export function ClientProvider({ children }: Readonly<{
        children: React.ReactNode;
      }>) {
        return (
          <ColorModeProvider config={}>
            <ResetStyles />
            <TypographyRoot />
            <CSSRoot />

            {children}
          </ColorModeProvider>
        );}`} />

</DocsSection>

### Usage

<DocsTextBlock className="m-b-medium-60">
  That's it! You can now use Foundation Design System.
</DocsTextBlock>

<DocsCodePreview
  code={`
    "use client"; // For nextjs apps

    import React from "react";
    import { Button } from "@usefui/components";
    import { useColorMode } from "@usefui/tokens";

    export default () => {
      const { colorMode, setColorMode } = useColorMode();
      const updateColorMode = () => {
        if (colorMode === "dark") setColorMode("light");
        else if (colorMode === "light") setColorMode("dark");
      };

      return (
        <Button sizing="small" variant="ghost">
          Color mode:&nbsp;{colorMode}
        </Button>
      );
    }`}

/>
