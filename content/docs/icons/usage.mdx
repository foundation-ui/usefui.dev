### Setup

<DocsTextBlock className="m-b-medium-60">
  The Icon component is built using the [Compound
  pattern](https://www.patterns.dev/react/compound-pattern/). By importing
  `Icon`, you can invoke every compound components without the need to import
  them all.
</DocsTextBlock>
