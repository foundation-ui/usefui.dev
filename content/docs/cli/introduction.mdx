<DocsSection>
  <DocsTextBlock className="m-b-medium-60">
    `@usefui/svgjsx` is designed to streamline your workflow when working with
    SVG files in modern JavaScript and TypeScript projects. Built with
    developers and product teams in mind, SVGJSX offers a fast, type-safe
    solution for converting, optimizing, and bundling SVG files into
    ready-to-use JSX components.
  </DocsTextBlock>
</DocsSection>

### Key features

<DocsSection>

#### Type Safety

<DocsTextBlock className="m-b-medium-60">
  Generate type-safe TypeScript components that integrate seamlessly with your
  existing codebase.
</DocsTextBlock>

#### Framework Agnostic

<DocsTextBlock className="m-b-medium-60">
  Compatible with any JSX-based framework such as Next.js, Electron, or Astro.
</DocsTextBlock>

#### Optimization

<DocsTextBlock className="m-b-medium-60">
  Automatically optimize SVG files for web use, reducing file sizes without
  compromising quality.
</DocsTextBlock>

#### Bundle Ready

<DocsTextBlock className="m-b-medium-60">
  Output components that are ready to be imported and used throughout your
  application.
</DocsTextBlock>

</DocsSection>

### Why SVGJSX?

<DocsSection>

#### Speed

<DocsTextBlock className="m-b-medium-60">
  Quickly convert and optimize SVG files with a simple command.
</DocsTextBlock>

#### Consistency

<DocsTextBlock className="m-b-medium-60">
  Maintain a single source of truth for all your visual assets.
</DocsTextBlock>

#### Flexibility

<DocsTextBlock className="m-b-medium-60">
  Works with your favorite frameworks and tools.
</DocsTextBlock>

<DocsTextBlock className="m-b-medium-60">
  Whether you're working on a new project or maintaining an existing one,
  `@usefui/svgjsx` helps you manage your SVG assets efficiently, ensuring
  consistency and reducing duplication.
</DocsTextBlock>

</DocsSection>
