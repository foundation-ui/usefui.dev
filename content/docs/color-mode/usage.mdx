#### Setup

<DocsTextBlock className="m-b-medium-60">
  `@foundation-ui` uses a custom Design Tokens Engine and requires a context and
  a structured design tokens values, make sure to have the `ColorModeProvider`
  context in your application setup.
</DocsTextBlock>

<DocsTextBlock className="m-b-medium-60">
  `ColorModeProvider` comes with a pre-built Design Token library set as
  default. You can use a custom design tokens library by passing it using the
  `config` property but it must be structured as defined in the type definition.
</DocsTextBlock>

<DocsTextBlock className="m-b-medium-60">
  Go to the `@foundation-ui/core` details to see how to generate your custom
  Design Tokens library.
</DocsTextBlock>

</DocsSection>
