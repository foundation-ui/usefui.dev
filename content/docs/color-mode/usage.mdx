### Get started with Color Mode

  Color Mode in available on `@usefui/tokens`, to use it, you first need to
  install the package:
</DocsTextBlock>

<DocsCodePreview language="shell" code={`pnpm add @usefui/tokens`} />
</DocsSection>

<DocsSection>
### Setup

<DocsTextBlock className="m-b-medium-60">
  `@usefui` uses a custom Design Tokens Engine and requires a context and a
  structured design tokens values, make sure to have the `ColorModeProvider`
  context in your application setup.
</DocsTextBlock>

<DocsTextBlock className="m-b-medium-60">
  `ColorModeProvider` comes with a pre-built Design Token library set as
  default. You can use a custom design tokens by passing it using the `config`
  property but it must be structured as defined in the type definition.
</DocsTextBlock>

<DocsTextBlock className="m-b-medium-60">
  Go to the `@usefui/core` details to see how to generate your custom Design
  Tokens library.
</DocsTextBlock>

</DocsSection>

<DocsSection>

  <DocsCodePreview
    code={`import { ColorModeProvider } from "@usefui/tokens";

      function Provider() {
        return (
          <ColorModeProvider config={/** */}>
            <Component />
          </ColorModeProvider>
        );
      }

      function Component() {
        const { colorMode, setColorMode } = useColorMode();

        return ( {/** Your code here **/} );
      }

`}/>

</DocsSection>
